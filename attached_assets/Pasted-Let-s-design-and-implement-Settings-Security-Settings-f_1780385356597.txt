Security Settings is an organization-level module responsible for authentication policies, password policies, session management, login protection, two-factor authentication enforcement, account security, and audit controls.

---

# Security Settings

## Password Policy

Configure password requirements.

### Settings

* Minimum Password Length
* Maximum Password Length
* Require Uppercase Letter
* Require Lowercase Letter
* Require Number
* Require Special Character
* Password Expiry Days
* Prevent Password Reuse
* Password History Count
* Force Password Change On First Login

### Example

```text
Minimum Length: 8
Require Uppercase: Yes
Require Number: Yes
Require Special Character: Yes
Password Expiry: 90 Days
Password History: 5 Passwords
```

---

## Login Policy

Control login security.

### Settings

* Maximum Failed Login Attempts
* Account Lock Duration
* CAPTCHA After Failed Attempts
* Allow Concurrent Logins
* Allow Multiple Devices
* Remember Me Duration
* Force Logout After Password Change

### Example

```text
Failed Attempts: 5
Lock Duration: 30 Minutes
CAPTCHA After: 3 Attempts
```

Manage user sessions.

### Settings

* Access Token Expiry
* Refresh Token Expiry
* Session Timeout
* Idle Session Timeout
* Maximum Active Sessions Per User
* Force Logout All Sessions
* Revoke Refresh Tokens

### Features

* View Active Sessions
* Terminate Session
* Terminate All Sessions

## Two-Factor Authentication (2FA) Policy

Organization-wide 2FA management.

### Global Settings

* Enable 2FA Feature
* Enforce 2FA

### Enforcement Modes

* Optional
* Mandatory For All Users
* Mandatory For Admin Roles
* Mandatory For Selected Roles

### Supported Methods

Phase 1:

* Email OTP

Future:

* Authenticator App (TOTP)
* SMS OTP
* Backup Codes

### Grace Period

* Immediate
* 3 Days
* 7 Days
* 15 Days
* 30 Days

### Recovery Options

* Allow Recovery Codes
* Allow Admin Reset
* Allow Backup Email

### Admin Actions

* Reset User 2FA
* Disable User 2FA
* Force User Reconfiguration
* View User 2FA Status

### User Status

* Enabled
* Disabled
* Pending Setup
* Locked
