Enhance the Client Settings module to become the central configuration hub for every tenant.

IMPORTANT

This is ONLY the Client Settings framework.

Do NOT implement business logic for Offer Letters, Email Sending, Approval Workflows or Integrations.

Create the module structure, database tables, APIs and UI only for the sections listed below.

==================================================
CLIENT SETTINGS
==================================================

This module is automatically provisioned whenever a new Client (Tenant) is created.

It is a mandatory system module.

Each tenant has its own isolated settings.
