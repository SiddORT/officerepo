Let's redesign the Recruitment Dashboard UI completely.

IMPORTANT

This is ONLY a UI/UX redesign.

Do NOT change database schema.
Do NOT change APIs.
Do NOT change business logic.

Only redesign the dashboard using the existing design system, components and dark theme already used across Office Repo.

The page should feel modern, premium and enterprise-grade while remaining extremely easy for a non-technical HR executive to understand.

==================================================

DESIGN GOALS

==================================================

• Clean and spacious layout

• Remove unnecessary clutter

• Dashboard should show information, not navigation

• Sidebar already provides navigation

• Follow existing Office Repo dark theme

• Consistent card radius, shadows and spacing

• Add subtle premium animations

• Responsive

• Professional typography hierarchy

Do NOT make it flashy.

Animations should be elegant.
